New condacts:

queueno/stackno = any value 1-4. If you need more stacks/queues edit stack_queue_lib.jsp file and change MAX_QUEUE or MAX_STACK value.

PUSH setno value => Add value to a stack
POP setno flagno => Pops value from a stack and stores it in the flag (last in, first out)

QPUSH setno value => Add value to a queue
QPOP setno flagno => Pops value from a queue and stores it in the flag (first in, first out)
